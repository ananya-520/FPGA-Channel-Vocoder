# FPGA-Channel-Vocoder
A 32-band FPGA channel vocoder developed using MATLAB, Simulink, and Xilinx System Generator for the Basys3 FPGA. Implements speech analysis and synthesis with IIR filter banks, fixed-point DSP, DDS oscillators, BRAM-based audio playback, UART communication, and FPGA resource optimization through hardware folding.

This project is created as a part of *ECTE433 - Embedded Systems* Project Requirements under Computer and Autonomous Systems Engineering course in UOWD.
